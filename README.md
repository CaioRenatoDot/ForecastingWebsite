# Weather Forecast Interface

Este projeto consiste na implementação de uma interface web baseada em um protótipo criado no Figma. O trabalho foi desenvolvido como atividade da disciplina de Interação Humano Computador (IHC).

A proposta do projeto é recriar a interface definida no design do Figma utilizando tecnologias modernas de desenvolvimento web, mantendo a organização visual e a estrutura do layout presentes no protótipo original.

## Objetivo

O objetivo principal é praticar o processo de desenvolvimento de interfaces a partir de um protótipo de design, aplicando conceitos de Interação Humano Computador. O projeto busca reproduzir visualmente a interface proposta no Figma e compreender como elementos de design são transformados em componentes funcionais.

Entre os pontos trabalhados no projeto estão:

* organização e hierarquia visual da interface
* consistência entre os elementos do layout
* clareza na apresentação das informações meteorológicas fazendo alimentação pela API
* separação de componentes na aplicação

## Tecnologias utilizadas

O projeto foi desenvolvido utilizando as seguintes tecnologias:

* React
* Tailwind CSS
* JavaScript
* HTML

O React foi utilizado para estruturar a aplicação em componentes reutilizáveis, enquanto o Tailwind CSS foi utilizado para a estilização da interface de forma utilitária e responsiva.

## Contexto acadêmico

Este projeto foi desenvolvido como atividade prática da disciplina de Interação Humano Computador (IHC). O trabalho tem como objetivo aplicar conceitos relacionados a design de interfaces, usabilidade e implementação de layouts a partir de protótipos.
